Integración en 1 línea

Copiá este snippet y pegalo antes del </body> de tu sitio. El chat aparece automáticamente.

<!-- FOMO Chat Widget -->
<script
  data-project="4SqKBrE3GDwfCRsdxVmYC"
  src="https://core.fomo.com.ar/webchat/widget.js"
></script>

¿Querés cambiar el color del widget? Configuralo desde el panel admin.

# Configurar webchat via API
curl -X PUT https://core.fomo.com.ar/api/v1/projects/TU_PROJECT_ID/webchat \
  -H "Authorization: Bearer TU_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Chat con nosotros",
    "welcomeMessage": "¡Hola! ¿En qué te puedo ayudar?",
    "primaryColor": "#7c3aed",
    "agentName": "Elena",
    "agentId": "TU_AGENT_ID",
    "enabled": true
  }'

¿Qué incluye el widget?

💬

Chat flotante

Bubble en esquina inferior derecha, no interrumpe la navegación del usuario.

🤖

IA de FOMO

Conectado directamente a tu agente configurado en el panel. Responde en segundos.

🎨

Personalizable

Color, nombre y mensaje de bienvenida configurables desde el panel admin.

📱

Mobile-friendly

Responsive por defecto. Se adapta a cualquier tamaño de pantalla.

💾

Sesiones persistentes

El historial de chat se guarda en localStorage para no perder el contexto.

Sin dependencias

JavaScript puro, CSS inline. No requiere React, Vue ni ningún framework.
